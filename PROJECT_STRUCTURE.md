## Описание пакетов

### src/core
Основные модули запуска и управления приложением. Содержит точки входа и класс `IdentityChecker`, который координирует все операции анализа профилей ВКонтакте.

### src/vk_api
Модули для взаимодействия с API ВКонтакте. Обеспечивает получение данных о пользователях, друзьях, фотографиях через официальное API VK.

### src/matchers
Модули для сравнения и сопоставления данных профилей. Каждый матчер отвечает за свой аспект сравнения:
- **name_matcher** - сравнение по имени и фамилии
- **friends_matcher** - анализ общих друзей
- **geo_matcher** - сравнение географических данных
- **demographics_matcher** - сравнение демографических показателей
- **visual_matcher** - визуальное сравнение фотографий
- **content_matcher** - анализ контента (посты, фото)
- **profile_comparer** - комплексное сравнение профилей

### src/face_recognition
Модули для распознавания и сравнения лиц с использованием различных библиотек и алгоритмов.

### src/handlers
Обработчики данных - промежуточный слой между API и логикой анализа. Отвечают за загрузку, обработку и сохранение данных.

### src/output
Генератор выходных данных - создает HTML-отчеты с результатами анализа профилей.

### src/utils
Вспомогательные утилиты - логирование, конфигурация, работа с файлами, подготовка данных.

## Соглашения об именовании

- **Файлы модулей**: snake_case (например, `vk_api_client.py`)
- **Классы**: PascalCase (например, `VKAPIClient`)
- **Функции и методы**: snake_case (например, `get_user_info`)
- **Константы**: UPPER_SNAKE_CASE (например, `MAX_FRIENDS_COUNT`)
- **Приватные методы**: начинаются с `_` (например, `_internal_method`)

## Запуск проекта

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск анализа профиля
python run.py <user_id или screen_name>

# Запуск проверки идентичности
python run_identity_checker.py
```

## Требования

- Python 3.8+
- Библиотеки для работы с VK API
- Библиотеки для распознавания лиц (OpenCV, MediaPipe)
- Библиотеки для генерации отчетов (Jinja2)
