**Добавлено:**

- Поддержка повторных попыток при ошибках сети
- Логирование через модуль logger.py

- Настраиваемые параметры (timeout, retries, delay)
- Обработка таймаутов

**Пример улучшений:**
```python
# Было

# Стало

    params=params, 
    timeout=self.timeout

```
