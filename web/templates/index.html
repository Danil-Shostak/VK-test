Возможности системы

Парсинг профилей:
  • Полная информация о пользователе (имя, дата рождения, город, образование, работа)
  • Список друзей с детальной информацией
  • Загрузка и сохранение фотографий
Сравнение профилей:
  • Анализ имени (точное, нечеткое, фонетическое)
  • Геолокационный анализ (город, страна, часовой пояс)
  • Анализ социальных связей (общие друзья, группы)
  • Демографический анализ (возраст, пол, образование)
  • Визуальное сравнение фотографий
Как начать
  1. Перейдите на страницу Парсинг
  2. Введите URL или ID профиля VK
  3. При необходимости укажите API токен
  4. Нажмите "Начать парсинг" и дождитесь завершения
