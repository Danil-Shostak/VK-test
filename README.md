## Описание проекта

VK Parser - это комплексный инструмент для анализа профилей ВКонтакте, включающий возможности парсинга данных профилей и систему идентификации/сравнения профилей для определения того, могут ли два профиля принадлежать одному человеку.

Проект написан на Python и использует официальный VK API для получения данных.

---

## Возможности

### Парсинг профилей
- ✅ Получение полной информации о пользователе (имя, дата рождения, город, страна, образование, работа и др.)
- ✅ Загрузка списка друзей с детальной информацией
- ✅ Скачивание всех фотографий пользователя
- ✅ Сохранение данных в различных форматах (JSON, CSV, TXT)
- ✅ Автоматическая генерация HTML-отчета с визуализацией

### Система идентификации профилей
- ✅ Комплексное сравнение профилей по множеству критериев
- ✅ Анализ имени (точное совпадение, нечеткое сопоставление, фонетический анализ)
- ✅ Анализ геолокации (города, страны, часовые пояса)
- ✅ Анализ социальных связей (общие друзья, группы)
- ✅ Анализ контента (интересы, стиль написания)
- ✅ Демографический анализ (возраст, пол, образование)
- ✅ Визуальное сравнение фотографий (опционально)

## Настройка

### Получение токена VK API

1. Перейдите на https://vk.com/apps?act=manage
2. Создайте новое приложение (Standalone)
3. Перейдите в "Настройки" → "Ключи доступа"
4. Скопируйте access_token

### Настройка токена

Токен можно настроить тремя способами (в порядке приоритета):

**Способ 1: Переменная окружения (рекомендуется)**
```bash
# Linux/macOS
export VK_TOKEN="ваш_токен"

# Windows (CMD)
set VK_TOKEN=ваш_токен

# Windows (PowerShell)
$env:VK_TOKEN="ваш_токен"
```

**Способ 2: Файл token.txt**
Создайте файл `token.txt` в корне проекта и поместите в него токен.

**Способ 3: config.py**
Отредактируйте файл `config.py` и укажите токен в переменной `VK_TOKEN`.

### Настройка логирования

В файле `config.py` можно настроить:
```python
LOG_LEVEL = 'INFO'  # DEBUG, INFO, WARNING, ERROR
LOG_FILE = 'vk_parser.log'  # Путь к файлу логов
```

## Использование

### Режим 1: Парсинг профиля

Запустите основной скрипт:
```bash
python main.py
```

Введите ссылку на профиль VK (например, `vk.com/id123456789` или `vk.com/durov`).

Результаты будут сохранены в папке `vk_results/`.

### Режим 2: Сравнение профилей (CLI)

```bash
# Интерактивный режим
python run_identity_checker.py

# Прямой вызов
python identity_checker.py -p1 "vk.com/id123" -p2 "vk.com/id456" -o result.json
```

Примеры использования:
```bash
# Сравнить два профиля по URL
python identity_checker.py -p1 "vk.com/id123456" -p2 "vk.com/id789012"

# Сравнить профиль из папки с профилем по URL
python identity_checker.py -p1 "vk_results/user1_20240301" -p2 "vk.com/id789012"

# Сравнить два профиля из папок
python identity_checker.py -p1 "vk_results/user1" -p2 "vk_results/user2"
```

## Опциональные модули

### Распознавание лиц

Проект поддерживает три библиотеки распознавания лиц (работают как fallback):

1. **face_recognition** (рекомендуется)
   - Наиболее точный метод
   - Требует установки dlib
   - Установка: `pip install face-recognition`

2. **OpenCV (Haar Cascade)**
   - Бесплатный и открытый
   - Менее точный, но быстрый
   - Установка: `pip install opencv-python`

3. **MediaPipe**
   - От Google
   - Требует загрузки моделей
   - Установка: `pip install mediapipe`

При отсутствии библиотек система работает без визуального сравнения.

## Ограничения

1. **Ограничения API VK**
   - Лимит запросов: 3 запроса в секунду
   - Некоторые данные могут быть недоступны для закрытых профилей

2. **Ограничения распознавания лиц**
   - Точность зависит от качества фотографий
   - Не работает с очень старыми или низкокачественными фото
   - Требует установки дополнительных библиотек

3. **Правовые ограничения**
   - Используйте только для законных целей
   - Соблюдайте политику конфиденциальности VK
   - Не сохраняйте и не распространяйте полученные данные без согласия
